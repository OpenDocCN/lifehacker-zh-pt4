# 海滨主屏幕

> 原文：<https://lifehacker.com/the-oceanfront-home-screen-1595574260>

Android:今天的特色主屏幕来自读者 [floridagizzi](http://floridagizzi.kinja.com/) 。该设计的特点是一个简化的三屏住宅，有工作、放松的空间，还有一个中央“开始”区。

Watch

该设计内置于 Nova Launcher。要在您自己的设备上获得这种外观，您需要:

*   [新星发射器](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher)T3】
*   [体素图标包](https://play.google.com/store/apps/details?id=com.benx9.palmtree)
*   [【uccw】](https://play.google.com/store/apps/details?id=in.vineetsirohi.customwidget)
*   [及时](https://play.google.com/store/apps/details?id=ch.bitspin.timely)
*   [最小 Any.do Widget](https://play.google.com/store/apps/details?id=com.gongchangstudio.minimalanydowidget)
*   [透明日历小工具](https://play.google.com/store/apps/details?id=net.daverix.TransparentCalendarWidget2)
