# 微妙的饲料主页屏幕

> 原文：<https://lifehacker.com/the-subtle-feed-home-screen-1716827246>

当你打开手机时，首先看到的就是你的主屏幕。你不妨利用一下那个空间。这种主屏幕设计在最小的主屏幕上放置了一个微妙的新闻提要，让您在不被侵入的情况下了解最新情况。



要在您的设备上实现这一设计，您需要:

*   [部件锁定器](https://play.google.com/store/apps/details?id=com.teslacoilsw.widgetlocker)
*   [顶点发射器](https://play.google.com/store/apps/details?id=com.anddoes.launcher)
*   [简单的 RSS 小工具](https://play.google.com/store/apps/details?id=de.j4velin.rssWidget)
*   [Compacticons 图标集](https://play.google.com/store/apps/details?id=drase.compacticons.iconpack)
*   [Zooper Widget Pro](https://play.google.com/store/apps/details?id=org.zooper.zwpro)
