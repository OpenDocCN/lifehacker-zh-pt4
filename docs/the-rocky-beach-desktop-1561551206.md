# 岩石海滩桌面

> 原文：<https://lifehacker.com/the-rocky-beach-desktop-1561551206>

Flickr 用户以下是如何让你的电脑看起来像这样。



Skarjoko 使用的是 [elementaryOS](http://elementaryos.org/) ，这是我们最喜欢的 Linux 发行版之一，尤其是在易用性和定制方面。许多设计线索来自于此，但也有一些其他的调整。以下是设置一切所需的内容:

*   壁纸，这是 eOS 的默认设置之一，所有这些 [你都可以在 Unixmen 这里](http://www.unixmen.com/download-elementary-os-luna-official-wallpapers/) 下载
*   图标的 [Numix 项目](http://numixproject.org/) 主题套件和应用启动器停靠在底部
*   [Conky](http://conky.sourceforge.net/) ，一款针对 Linux 系统的轻量级系统监视器
*   Conky 的皮肤让 Conky 在桌面上显示天气

这就是全部了。这款桌面的漂亮之处在于 eOS 的总体外观和感觉，而 Numix 以犀利的方式为桌面增添了一些鲜明的视觉效果。如果你正在使用 eOS，或者只是在考虑 Linux，这是一个很好的开始方式。如果你对设置感到好奇或者有其他问题，请访问 Skarjoko 的 Flickr 页面，查看并提出你的问题！
