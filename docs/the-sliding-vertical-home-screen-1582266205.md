# 滑动垂直主屏幕

> 原文:[https://life hacker . com/the-sliding-vertical-home-screen-1582266205](https://lifehacker.com/the-sliding-vertical-home-screen-1582266205)

我的彩屏用户 Peszek 没有用你最常用的应用程序创建一堆屏幕，而是创建了一个简单的主屏幕，有四列，可以扩展显示应用程序和信息。

Watch

其核心是，这个主题类似于主屏幕的概念，但更简单一些——只需点击一个类别，它就会向你显示该类别的应用程序，以及一个包含该类别信息的弹出窗口小部件(如当前正在播放的歌曲，或你的下一首即将播放的歌曲

垂直主屏幕是 Themer 应用程序的自定义设计。您可以下载。zip 文件包含来自下面的源链接的主题。要安装它，请遵循以下说明:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“我的主题”下，选择“导出”
4.  选择“垂直”
