# 橙色主屏幕的痕迹

> 原文：<https://lifehacker.com/the-trace-of-orange-home-screen-1592286502>

Android:今天的主屏幕把你需要的一切都放在几个主屏幕上。除了底部通常的稳定的快捷方式，这个主题还包括你最常用和最喜欢的应用程序的空间。



这个设计是 Themer 应用程序的自定义主题。要在您的设备上安装，请下载。zip 文件，然后按照以下步骤操作:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“我的主题”下，选择“导出”
4.  选择“orange.zip 的踪迹”
