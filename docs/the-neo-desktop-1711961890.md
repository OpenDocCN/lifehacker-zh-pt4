现在，如果你有一台苹果电脑，并想做一些这样的东西供你使用，你需要的东西如下:

*   [壁纸](https://www.flickr.com/photos/sanjeevneo/18330861299/in/pool-lifehacker-desktop-showandtell/)
*   用于 OS X 的 [GeekTool](http://projects.tynsoe.org/en/geektool/) 系统监视器和配置实用程序

Sanjeev 有一些定制的极客小程序，用于时间、日期和 Spotify“正在播放”,出现在他的壁纸元素周围——你必须前往他的 Flickr 页面(链接如下)才能获得它们——但他包括了你需要的代码，让它们在一个地方工作并出现在你的桌面上，这很棒！

我们可能会丢弃桌面上多余的文件夹，可能会自动隐藏 Dock，但显然 Sanjeev 有工作要做，我们不会因此责备他。如果你喜欢你所看到的，让他知道在下面的链接，如果你有问题或不能让事情工作，问他那里！
