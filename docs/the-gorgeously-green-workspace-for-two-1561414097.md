# 华丽的绿色双人工作空间

> 原文：<https://lifehacker.com/the-gorgeously-green-workspace-for-two-1561414097>

一些家庭办公室不会让你觉得被困在室内，而是有一个开放，微风习习的氛围，这要归功于照明，浅色墙壁和天然材料。两株植物、编织椅和一张自制的双人木桌构成了这个令人振奋的工作空间。



它属于设计和摄影二人组朱莉娅和塞维多夫·曼奇克。Yuri 建造了简单的书桌，两株植物(加上其他绿色配件)构成了空间。拿我来说，我想为自己收藏一张桌子，让我的伴侣拥有自己的，但是如果你在一个小空间里很舒服，并且愿意分享，这是一个很好的方式。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间展示和讲述](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) Flickr 池中，与我们分享。确保您包含的任何照片至少为 640x360。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
