# 云林桌面

> 原文：<https://lifehacker.com/the-forest-of-the-clouds-desktop-1726509106>

[apagajewski](https://www.flickr.com/photos/124439106@N04/) 的 linux 桌面提醒我们即将到来的寒冷的冬天早晨。清脆爽口，即使还在我们所在的暖侧。他是这样设置的。



自从我们推出 linux 桌面已经有一段时间了，但是这个看起来很不错。他运行的是 Arch Linux 的发行版 [Apricity OS](https://apricityos.com/) 。这也很容易设置。如果你想复制这种造型，你需要做的是:

*   [隧道景色，约塞米蒂壁纸](https://www.flickr.com/photos/rjshade/13554962825/) ，由拍摄[罗比遮挡](https://www.flickr.com/photos/rjshade/)
*   [Conky](http://conky.sourceforge.net/) ，一款针对 Linux 系统的轻量级系统监视器
*   [Conky——明日](http://zagortenay333.deviantart.com/art/Conky-Tomorrow-396034879) 主题为 Conky
*   [北极先验](https://github.com/agajews/ArcticApricity) 侏儒外壳主题
*   屏幕下方的 [Dash to Dock](https://extensions.gnome.org/extension/307/dash-to-dock/) GNOME 扩展为 dock launcher
*   [Numix 圈](https://numixproject.org/) 图标包
*   [Vimix 图标主题](http://gnome-look.org/content/show.php/Vimix-Icon-Theme?content=169873) 为状态图标

最终的结果看起来非常清晰，我们以前也见过一些很棒的 linux 桌面。如果你正在定制你自己的，有问题，需要帮助让一切看起来正确，或者只是想告诉 apagajewski 你有多喜欢他的作品，去他的 Flickr 页面(链接如下)让他知道是我们送你的！
