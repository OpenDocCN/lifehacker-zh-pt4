# 宁静主屏幕

> 原文：<https://lifehacker.com/the-serenity-home-screen-1715516777>

不， [不是那个](http://www.imdb.com/title/tt0379786/) 。这个主屏幕设计旨在给你忙碌的主屏幕带来一些平静。它包括几个最小的部件和一个中性的灰色配色方案。

Watch

这个主屏幕是为 Themer 和 Nova launcher 设计的。对于 Nova，以下是您需要的:

*   [新星发射器](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher)T3】
*   [Zooper Widget Pro](https://play.google.com/store/apps/details?id=org.zooper.zwpro)
*   [媒体实用程序](https://play.google.com/store/apps/details?id=com.batescorp.notificationmediacontrols.alpha)
*   [文件](https://www.dropbox.com/sh/rht4wd4gq8c4vgo/AABACrKA0-Xi_t4pHdaULn_va?dl=0)

要在主题上获得这种设计，请下载。zip 文件，然后按照以下步骤操作:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“类别”下，选择“导出”
4.  选择“Serenity.zip”
