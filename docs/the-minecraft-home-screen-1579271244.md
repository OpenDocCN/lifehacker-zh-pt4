# 《我的世界》主屏幕

> 原文:[https://lifehacker.com/the-minecraft-home-screen-1579271244](https://lifehacker.com/the-minecraft-home-screen-1579271244)

安卓:《我的世界》是一款游戏/创意工具，在释放你的想象力方面几乎可以与乐高 相媲美。自然，它非常适合您的主屏幕。

Watch

《我的世界》的主屏幕是 Themer 应用程序的定制设计。您可以下载。zip 文件包含来自下面的源链接的主题。要安装它，请遵循以下说明:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“我的主题”下，选择“导出”
4.  选择“《我的世界》[1]”
