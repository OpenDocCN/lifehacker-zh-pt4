# 白色冬季主屏幕

> 原文：<https://lifehacker.com/the-white-winter-home-screen-1685549256>

有些地方你可能看不出来，但我们现在正处于隆冬。如果你需要窗外的提醒，这个宁静的白色主屏幕会让你感受到冬天的寒意。



这个设计是为 Themer 和 Zooper 小部件设计的。您可以从下面的源代码链接下载 Themer 文件，并按照以下步骤进行安装:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“我的主题”下，选择“导出”
4.  选择“White-Winter-Themer.zip”

如果你想使用不同的启动器，设计者在源代码链接中有说明。这是你需要的:

*   [新星发射器](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher)T3】
*   [Zooper Widget Pro](https://play.google.com/store/apps/details?id=org.zooper.zwpro)

*   [源文件](http://www.android-tuning.sk/wp-content/uploads/White-Winter.zip)
