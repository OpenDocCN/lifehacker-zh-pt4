# Microbus 桌面

> 原文:[https://lifehacker.com/the-microbus-desktop-1602301935](https://lifehacker.com/the-microbus-desktop-1602301935)

我们之前已经强调过几款 [joergermeister](https://www.flickr.com/photos/joergermeister/) 的桌面，但是这款写满了“坐上派对大巴，上路吧”的桌面很平，看起来很简洁。以下是他如何设置的，以及你自己设置时需要的东西。

Watch

我们喜欢整个东西的外观——无边框、扁平的形状，定制的浏览器窗口和开始菜单增添了魅力。它并不适合所有人，但如果你喜欢它的外观，这里是你需要的:

*   威廉·坎宁安的 [大众巴士壁纸](http://simpledesktops.com/browse/desktops/2011/nov/26/vw-bus/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [Google Now 皮肤](http://joergermeister.deviantart.com/art/Google-now-410080356) 为雨量计为时钟和天气显示
*   任务栏和视觉风格图标的 [FCOAR 2014](http://fcoar.neiio.ca/) 图标包
*   Windows 8.1 的嘘视觉风格，彻底改变了资源管理器窗口的外观
*   [开始菜单](http://www.reviversoft.com/start-menu-reviver/) 为自定义开始菜单

这是项目的主要部分。这一切都很好地结合在一起，虽然它肯定不会是每个人的口味，但它是原创和有趣的，这是 Windows 8.1 开箱后所没有的。如果你喜欢你所看到的，或者遗漏了细节，去 joergermeister 的 Flickr 页面(链接如下)问你的问题吧！让他知道是我们派你去的。
