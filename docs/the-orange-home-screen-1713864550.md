# 橙色主屏幕

> 原文：<https://lifehacker.com/the-orange-home-screen-1713864550>

每个人的饮食中都需要一点维生素 C。当然，你不会从这个主屏幕上看到任何东西，但也许大量使用黑底橙会提醒你喝更多的果汁。或者看更多的网飞原创节目。两者之一。



要在您的设备上获得这种外观，您需要:

*   [新星发射器](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher) (或者另一个备用发射器)
*   [Zooper Widget Pro](https://play.google.com/store/apps/details?id=org.zooper.zwpro)
*   [媒体实用程序](https://play.google.com/store/apps/details?id=com.batescorp.notificationmediacontrols.alpha)
*   [橙色 Zooper Widget 皮肤](https://play.google.com/store/apps/details?id=com.behemoth.zwskin.orange) (说明也可以在这个列表中找到)

你有没有一个很棒的、被调整到遗忘的家或者你自己的锁屏，你想分享吗？把它贴在下面的评论里，或者贴在你自己的 Kinja 博客上，标签是“主屏幕展示”(没有引号)。一定要包括你是如何制作它的描述，这样我们就可以把它作为下一个特色主屏幕。
