# Eclipse 桌面

> 原文：<https://lifehacker.com/the-eclipse-desktop-1700956968>

读者 [普什卡](http://pushkardesai999.kinja.com/) 用几个容易配置的组件，在短短几分钟内就把这个刻板、开放、干净的桌面组装起来。他是这样做的。



普什卡将这个桌面提交到我们的 [桌面展示区](http://kinja.com/tag/desktop-showcase) ，如果你想在这里看到你的桌面，这是你应该做的！他使用的是 Windows，这一点并不明显，因为没有任务栏、系统托盘或其他清晰的菜单。所有的东西都整齐地藏起来，直到他需要的时候。如果你想让你的 Windows 机器也有同样的外观，这里是你需要的:

*   [月食壁纸](http://simpledesktops.com/browse/desktops/2014/dec/17/eclipse/) by [刘芷萌阿迪雅](http://jandyaditya.deviantart.com/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计的 [弧线主题](http://kaelri.deviantart.com/art/Arcs-112670528) 在屏幕中央创建时钟
*   雨量计的 [简单媒体皮肤](http://lilshizzy.deviantart.com/art/Rainmeter-Simple-Media-1-1-212678796) 用于侧面的日期和时间显示
*   屏幕左侧的应用启动器
*   火箭码头的 [乌瑞亚图标包](http://neiio.deviantart.com/art/Ourea-Icons-339972848)

这就是所需要的，你可以看到每个皮肤和主题在成品中的应用。如果你有问题，或者想知道更多关于 Pushkar 是如何做到这一切的，去他的 Kinja 博客(链接如下)问你的问题。让他知道我们有多喜欢他的作品！
