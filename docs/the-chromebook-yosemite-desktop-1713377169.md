# Chromebook Yosemite 桌面

> 原文：<https://lifehacker.com/the-chromebook-yosemite-desktop-1713377169>

保罗的桌面可能看起来像是 Mac，但实际上是在他的 Chromebook Pixel 上运行的。通过一点点定制和一些新图标，他把它变成了自己的，看起来很棒。这是他如何调整的。



记住，如果你想在这里展示你的桌面，请提交到我们的 [**桌面展示区**](http://kinja.com/tag/desktop-showcase) ！只需将它(以及你是如何制作的)发布到你的 Kinja 博客(你的账户附带的博客)上，并标记为“桌面展示”我们会看到它，仔细检查它，如果我们喜欢它，我们会特写它！你也可以像 Paul 在这里做的那样把它发布到我们的 Flickr pool 上，但是我们先检查桌面展示区！

也就是说，以下是你需要的:

*   约塞米蒂的壁纸，与约塞米蒂的 捆绑在一起的 [壁纸，却可以单独下载](http://9to5mac.com/2014/08/18/here-are-all-of-os-x-yosemites-beautiful-new-wallpapers/)
*   用 [彩陶图标包](http://tiheum.deviantart.com/art/Faience-icon-theme-255099649) 来替换系统图标
*   [GNOME](https://www.gnome.org/) 桌面环境，哪个 [能在 Chromebook](http://lifehacker.com/how-to-install-linux-on-a-chromebook-and-unlock-its-ful-509039343) 上运行，信不信由你
*   Linux 的 [变种](http://peterlevi.com/variety/) 壁纸更换器，它也在屏幕右下角创造了美丽的桌面时钟

大概就是这样。这里真正的问题是让 GNOME 在你的 Chromebook 上运行，这是这里其他一切的关键。如果你喜欢你所看到的，一定要去保罗的 Flickr 页面让他知道，如果你有问题，就提出来！
