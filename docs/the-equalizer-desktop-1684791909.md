# 均衡器桌面

> 原文：<https://lifehacker.com/the-equalizer-desktop-1684791909>

Flickr 用户 [伊丽莎白](https://www.flickr.com/photos/lareina1045/) 有一个非常漂亮的桌面，可以把她的音乐放在最前面和最中心。另外，它真的很容易设置。她是这样做的。



这个桌面可能看起来很复杂，但它实际上是一个单一的皮肤和一个实心壁纸。以下是你将它变成你自己的所需要的:

*   [壁纸](http://www.desktopbackgrounds.me/cold-blurred-colors/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   中央的 [桌面++雨量计的皮肤](http://fiftyniner.deviantart.com/art/Desktop-v1-1-2-493372796) 为正在播放的显示，系统监视器在“翅膀”和均衡器覆盖上

这就是这个的全部内容。那一层皮很结实，看起来很棒。如果你不确定如何让你的桌面也这样工作，去伊丽莎白的 Flickr 页面(链接如下)问你的问题，让她知道你有多喜欢她的桌面！
