# 简约的 IT 日常包

> 原文：<https://lifehacker.com/the-minimalist-it-everyday-bag-1578502817>

仅仅因为你身处其中，并不意味着你必须一直带着一大堆小玩意。事实上，对于读者尼古拉斯·摩尔来说，垃圾越少越好。



摩尔把所有东西都放在一个 [赫歇尔突击测验袋](http://shop.herschelsupply.com/collections/pop-quiz-backpack/products/pop-quiz-backpack-red) 里。这是里面的内容:

*   [MacBook Air I7 8GB 2013 年秋季](https://www.apple.com/macbook-air/)
*   [视网膜 iPad mini 2013 年秋季](https://www.apple.com/ipad/)
*   [歇尔包](http://store.apple.com/us/product/HC370VC/A/herschel-supply-co-pop-quiz-plus-backpack?fnode=50&fs=f%3Dherschel_supply%26fh%3D36c2%252B32e8)
*   紧凑型工具包\手电筒
*   8GB 闪存驱动器
*   NT 离线 Windows 密码破解 usb 拇指驱动器
*   笔记本
*   最喜欢的钢笔(木头来自托马斯·杰斐逊在蒙蒂塞洛种植的一棵树，左边的树在旧的 2 美元钞票的背面)
*   [肖恩约翰眼镜](http://www.seanjohn.com/)T3】
*   联系人案例
*   [MagSafe 2](http://store.apple.com/us/product/MD565LL/A/apple-60w-magsafe-2-power-adaptermacbook-pro-with-13-inch-retina-display?afid=p219%7CGOUS&cid=AOS-US-KWG-PLA)
*   未图示:Lightning、USB 迷你线、USB 以太网适配器。
*   商业名片
