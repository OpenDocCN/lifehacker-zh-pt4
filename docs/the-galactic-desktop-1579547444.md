# 银河桌面

> 原文:[https://lifehacker.com/the-galactic-desktop-1579547444](https://lifehacker.com/the-galactic-desktop-1579547444)

读者[ninjachharlie](http://ninjacharlie.kinja.com/)使用 GeekTool 在工作时将他的 Twitter 信息和他正在听的歌曲嵌入到他的壁纸中，因此他不需要另一个应用程序来跟踪它们。以下是你也可以做到的方法。

ninjacharlie 将他的桌面提交到我们的 [桌面展示](http://kinja.com/desktop-showcase) ，如果你想在这里看到你的设置，这是你应该做的！你所需要做的就是把一个截图贴在你的 Kinja 博客上(你的评论账号附带的那个)，同时附上你用来定制它的壁纸、小工具和其他工具。

以下是让你的 OS X 桌面看起来像这样所需的内容:

*   DeviantArt 上 Starkiteckt 的 [仙女座壁纸](http://starkiteckt.deviantart.com/art/Andromeda-451971383)
*   用于 OS X 的 [GeekTool](http://projects.tynsoe.org/en/geektool/) 系统监视器和配置实用程序，用于所有嵌入式项目(时间、推特和正在播放)
*   [T Twitter 命令行界面](https://github.com/sferik/t)
*   这些 [用于在 GeekTool/Nerdtool](http://www.leancrew.com/all-this/2011/07/spotify-info-on-the-desktop-via-nerdtool/) 中设置 Spotify“正在播放”的脚本

这就是全部了。GeekTool 可能是一个资源耗尽的工具，但是这个设置很简单，也很聪明，并且把事情减到最少。最重要的是，ninjacharlie 不需要让 spotify 在桌面上徘徊来查看正在播放的内容，也不需要打开 Twitter 应用程序来查看他的流中正在发生的事情。因为他使用的是命令行界面，所以只要他愿意，他可以打开一个终端窗口(或者一个新窗口，如果他已经在终端中工作的话)来回复或者与人互动。如果你喜欢你所看到的，去他的 kinja 博客(链接如下)让他知道你喜欢他的作品！如果你遇到了麻烦， [查看我们的 GeekTool 指南，获取提示](https://lifehacker.com/build-an-attractive-informative-mac-desktop-with-geekt-5834676) 。
