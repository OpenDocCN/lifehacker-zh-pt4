# 优胜美地桌面

> 原文：<https://lifehacker.com/the-very-yosemite-desktop-1651466749>

读者 [joshnroy](https://www.flickr.com/photos/124027196@N04/) 刚刚升级到约塞米蒂，做了一点小小的调整，提交了这个运行在 OS X 约塞米蒂上的约塞米蒂国家公园宁静的桌面。这是他用来设置的。

Watch

实际上，乔森罗伊的桌面配置非常简单——它不使用大量工具，而且大部分都可以在约塞米蒂找到。如果您想让新升级的约塞米蒂苹果电脑看起来像这样，以下是您需要的:

*   壁纸实际上是内置在 Yosemite 中的，在新的 wallpapers 文件夹中。如果你没有或者只是想下载，可以 [在这里](http://9to5mac.com/2014/08/18/here-are-all-of-os-x-yosemites-beautiful-new-wallpapers/) 抢到它和其他新的优胜美地壁纸。
*   Yosemite 的“黑暗模式”，将菜单栏和 dock 调暗(在“系统偏好设置”>“通用”中提供)
*   [über sicht](http://tracesof.net/uebersicht/)，之前我们已经介绍过的 OS X 定制套件。
*   [简单时钟小部件](https://github.com/soberstadt/simple-clock-widget) 为 bersicht

就是这样！关于 bersicht 的一件好事是，如果你需要的话，有更多的小部件和脚本可以用来定制你的桌面，所以这只是一个开始。在我们的测试中，bersicht 占用的系统资源也很少——尤其是与 GeekTool 相比——所以你可以随心所欲地定制内容。如果你对这个桌面或者 joshnroy 是如何制作的有任何疑问，请访问他的 Flickr 页面，链接如下，提出问题！
