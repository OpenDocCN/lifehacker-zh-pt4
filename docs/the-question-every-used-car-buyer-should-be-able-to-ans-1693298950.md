# 每个二手车买家都应该能够回答的问题

> 原文:[https://jalopnik . com/the-question-every-二手车买家应能-ans-1693298950](https://jalopnik.com/the-question-every-used-car-buyer-should-be-able-to-ans-1693298950)

平均来说，我每天会接到五个电话，是那些购买了有缺陷的二手车并寻求法律帮助的人打来的。我问的问题是哪一个最难？它有保修吗？ 不知道这个问题的答案就好比不知道汽车是否是带发动机出售的。

我经常写关于汽车和法律的文章，因为我在密歇根州实践柠檬法则。一般来说，这包括购买全新的有缺陷的汽车。所以当有人打电话到我的办公室时，我问的第一个问题是，“你的车是新买的吗？”如果他们说“不”，我会问，“它有保修吗？”许多人对此的回答是“我不知道”

一些后续的问题通常会确定下来。这些车辆是按原样出售的，并且没有售后维修方面的保修。虽然我以前写过一些关于这些问题的文章，但我认为改变许多人的购车方式是很重要的。消费者需要:1)购买保修(连同汽车)或者 2)根据缺少保修的情况给汽车定价。

没有维修或更换保修或服务合同的二手车的价值远远低于有保修或服务合同的二手车。有人说，“但是我花了 30，000 美元买的！”好像价格是某种保证。一辆汽车可能要 30，000 或 300 美元；没关系。按原样出售，没有维修或更换保修，也没有服务合同，价格不是汽车质量的指标。这只是你付出的代价。

二手车可以有几种不同的“修理或更换”保证。新款二手车可能会附带剩余的工厂保修。在我看来，这是你用二手车能得到的最好的东西。你知道谁支持保修，你可以把它拿到不止一个地方修理，你也知道它的条款是什么。这些车会更贵，但它们更值钱。

二手车经销商可能会为你提供延长服务合同。许多人将这个术语与“保修”互换使用，但还是有一些区别。ESC 的覆盖范围通常更有限，通常由汽车制造商以外的人支持，条款可能相当有限。在你购买其中一个之前，阅读它的条款，看看它包括什么和不包括什么。有些是如此有限，使他们虚幻的。也就是说，由于其令人发指的排除，它们并没有真正以有意义的方式涵盖任何内容。

一些制造商提供延期服务合同，这些合同比其他制造商出售的合同更受欢迎，即所谓的“第三方”服务合同。

一些经销商自己提供保修服务，他们承诺如果汽车在一定的时间或里程内以及在一定的情况下发生故障，他们会修理汽车。这些和经销商的信誉一样好，不多不少。

所以，这里是你在购车时需要做的。找一辆你喜欢的车。询问它是否有保修。如果他们说没有，那么可以推测它是按原样出售的。相应地定价，要知道在你从车中得到任何用途之前，发动机可能会爆炸，变速器可能会失灵。这种风险是你所购物品的一部分，所以要考虑进去。

或者，继续找，直到你找到一辆有保修的待售汽车。当他们说是时，问是什么样的保修？是制造商保修、延长服务合同还是经销商保修？它的条款是什么？ [又写在哪里？](https://oppositelock.kinja.com/the-one-thing-you-should-know-when-you-buy-a-car-1611470939)T3】

延长服务合同通常会让你花钱。阅读覆盖范围，知道你可能可以在 ESC 上协商价格。他们想从第三方获得 595 美元的 ESC？如果他们和你平摊费用，就主动提出买车。最坏的情况是他们说不。

这里的要点是，你得到的汽车保修或服务合同是有价值的——和知道汽车是否能行驶一样重要。大多数人不会在没有至少看到它能跑的情况下买车。投入同样多的精力来确定汽车的覆盖范围。太多的人听到销售人员说“如果你的车有任何问题，把它带回来，我们会帮助你，”并认为这是某种形式的保修。它不是。如果你不确定，再问一次:**我的这辆车可以得到什么样的保修，书面保证在哪里？**
