Jayesh 将他的桌面提交到我们的 [桌面展示区](http://kinja.com/desktop-showcase) ，如果你想在这里看到它的高亮显示，这是你应该做的！只需将它(以及你是如何制作的)发布到你的 Kinja 博客(你的账户附带的博客)上，并标记为“桌面展示”我们会看到它，仔细检查它，如果我们喜欢它，我们会特写它！

但是，如果您喜欢这个设置，您需要将它变成您自己的设置:

*   [冷宫壁纸](http://arand4.deviantart.com/art/Limbo-basic-lines-wallpaper-344317821)来自 DeviantArt
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计 的 [Enigma 套件，用于创建笔记、任务、Gmail 阅读器和 Chrome 图标部分](http://enigma.kaelri.com/downloads/)
*   [为雨量计的编码皮肤](http://lilshizzy.deviantart.com/art/Rainmeter-Encoded-v1-2-3-215992941) 为日期和时间显示
*   搜索栏的 [雨量计的 Omnimo 皮肤](http://omnimo.info/download.html)——可以通过键盘快捷键在谷歌、维基百科、Wolfram Alpha 等之间切换
*   [ObjectDock](http://www.stardock.com/products/ObjectDock/) 带有 [烟雾背景](http://lifehacker.com) 作为应用程序启动器(尽管我们确信你可以用 RocketDock 做类似的事情)
*   [Ecqlipse 2 图标包](http://chrfb.deviantart.com/art/ecqlipse-2-quot-ico-quot-66339090) 改变 dock 图标

这就是你所需要的。你可以在 Jayesh 的 Kinja 博客上看到更多桌面操作的照片，链接如下。
