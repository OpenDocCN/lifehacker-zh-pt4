整个事情是一个相当简单的雨量计设置，应该只需要你几分钟，一旦你有它的所有组件。这是你需要的:

*   壁纸， [【云存储】Satchell Drakes](http://simpledesktops.com/browse/desktops/2010/dec/01/cloud-storage/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [简单句三皮为雨量计](http://white-baron.deviantart.com/art/Simplesentencethree-175994306) 创建 CPU 和内存监视器
*   [KitKat 时钟皮肤](http://rainmeterhub.com/skins/kitkat-clock/) 在屏幕中央创建时钟
*   桌面搜索栏的 [Googlebar 皮肤](http://rainmeterhub.com/skins/googlebar/) 为雨量计
*   用于雨量计的 [Zune 播放器用于音乐播放器的](http://maxvanijsselmuiden.deviantart.com/art/Zune-Player-for-Rainmeter-195254923)
*   将 [令牌图标设置为](http://brsev.deviantart.com/art/Token-128429570) 来替换默认任务栏图标
*   坎布里亚的 [启动宝珠](http://www.deviantart.com/art/Cambria-Start-Orb-199865350) 取代了启动按钮

这就是你所需要的，而且他们都很独立。一旦安装并配置到您想要的地方，剩下的事情就变得轻而易举了。如果您对它的设置有任何疑问，或者需要帮助，请访问 Richard 的 Flickr 页面(链接如下),提出您的问题——并让他知道我们邀请了您！
