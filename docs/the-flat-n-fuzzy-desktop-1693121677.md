# 平坦模糊的桌面

> 原文：<https://lifehacker.com/the-flat-n-fuzzy-desktop-1693121677>

读者 [皇家芝士](http://royalewcheese.kinja.com/) 爱 [平板台式机](http://lifehacker.com/the-owl-desktop-1527365896) ，我们也一样。这一次他的桌面向传说中的 fuzz 踏板致敬。如果你知道要找什么，这种简单、平淡的造型就能说明很多问题。这是所有设置的方式。



音乐家和音乐爱好者可能甚至不需要我告诉你那是什么样的踏板来让你识别它，这是这个桌面如此棒的一部分原因。另一件事是，这是一个 Linux 桌面，我们已经有一段时间没有看到太多了。无论哪种方式，您都需要将自己的 Linux 系统做成这样:

*   [壁纸](http://byronfortescue.deviantart.com/art/Big-Muff-157005843)
*   [平淡主题](http://cinnamon-spices.linuxmint.com/themes/view/142) 为肉桂
*   [天秤座](http://gnome-look.org/content/show.php/Libra?content=167689)主题为 GTK3
*   屏幕下方的 [Docky app 启动器](http://wiki.go-docky.com/index.php?title=Installation) 为 dock
*   [Ulta 平面图标包](http://gnome-look.org/content/show.php/Ultra-Flat-Icons?content=167477) 替换 dock 和系统图标

这就是这个的全部内容！如果你已经在运行上面有 [肉桂](http://cinnamon.linuxmint.com/) 的 Linux(比如 Linux Mint)，那么剩下的对你来说应该很容易。如果你有关于如何设置的问题，以及如何设置这样的桌面，请访问他的 Kinja 博客(链接如下),阅读更多关于如何设置的信息，并提出你的问题。
