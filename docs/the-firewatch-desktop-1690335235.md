# Firewatch 桌面

> 原文:[https://lifehacker.com/the-firewatch-desktop-1690335235](https://lifehacker.com/the-firewatch-desktop-1690335235)

在 Flickr 上， [艾哈迈德](https://www.flickr.com/photos/130314598@N05/) 为 [火眼金睛](http://www.firewatchgame.com/) 而欢呼，这是一款即将到来的游戏，有望令人惊叹。他的桌面让我们从一个美丽的制高点看日落——但它也帮助他启动应用程序和检查他的系统状态。他是这样设置的。

Watch

我们可以想出更糟糕的方法来等待一个看起来很棒的新游戏的发布，而不是把你的桌面献给它(还记得 [命运桌面](http://lifehacker.com/the-destiny-desktop-1605919811) ？)

这款桌面专用于 [Firewatch](http://www.firewatchgame.com/) ，这是一款来自 [Campo Santo](http://www.camposanto.com/) 的即将推出的独立游戏，它从宣传片中汲取了一些艺术元素，并将其转换为一个精美的 HUD，你可以坐下来立即查看正在发生的事情，或者可以快速最小化以查看系统状态或正在播放的内容——但它也不会妨碍 windows 的运行和工作时间。以下是让它成为你自己的所需要的:

*   [火眼壁纸](http://www.1080p-wallpapers.com/fire-watch/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计的 [VisBubble 皮肤](http://madhoe.deviantart.com/art/VisBubble-for-Rainmeter-488601501) 创建圆形音频可视化器
*   雨量计的 [线 2.0 皮肤](http://rainmeterhub.com/skins/lines-2-0/) 为系统监视器(RAM/CPU/etc)
*   雨量计的 [iTunes 播放器皮肤](http://rainmeterhub.com/skins/itunes-player/) 显示“正在播放”图像
*   雨量计的 [Emiko 发射器](http://rainmeterhub.com/skins/emiko-launcher/) 底部中央为九点式发射器

这就是这件作品的全部，所有的部分看起来都非常吻合。如果你在设置上有困难，或者有问题，请访问 Ahmad 的 Flickr 页面(链接如下),询问他是如何做到这一切的——并让他知道是我们派你来的！
