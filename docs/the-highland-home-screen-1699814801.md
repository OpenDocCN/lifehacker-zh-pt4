# 高地主屏幕

> 原文：<https://lifehacker.com/the-highland-home-screen-1699814801>

如果你是山景和宁静色彩的爱好者，这个主屏幕就是为你准备的。该设计采用背景图像中使用的颜色，给用户界面一个柔和的色调。

Watch

这个主屏幕是为带有 Zooper 的 Nova Launcher 设计的。要在您的手机上实现这一设计，您需要:

*   [新星发射器](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher) (或其他自定义发射器)
*   [Zooper Widget Pro](https://play.google.com/store/apps/details?id=org.zooper.zwpro)
*   [Tasker](https://play.google.com/store/apps/details?id=net.dinglisch.android.taskerm) (用于 Whatsapp/Hangouts 通知)
*   [媒体实用程序](https://play.google.com/store/apps/details?id=com.batescorp.notificationmediacontrols.alpha)
*   [高地主题](https://play.google.com/store/apps/details?id=com.androidtuning.zwskin.Highland)
