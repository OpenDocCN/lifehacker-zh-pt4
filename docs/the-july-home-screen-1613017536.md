# 七月主屏幕

> 原文:[https://lifehacker.com/the-july-home-screen-1613017536](https://lifehacker.com/the-july-home-screen-1613017536)

为了简单，总有话要说。这种主屏幕设计当然也是如此。这一款的特点是在当前日期和时间旁边有一个你一天的活动的小概览，还有几个通知徽章，让你知道你需要处理多少封电子邮件或未接来电。

Watch

该设计是主题的自定义皮肤。您可以从下面的源代码链接下载它，并按照以下说明进行安装:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“我的主题”下，选择“导出”
4.  选择“July.zip”

你有没有一个很棒的、被调整到遗忘的家或者你自己的锁屏，你想分享吗？把它贴在下面的评论里，或者贴在你自己的 Kinja 博客上，标签是“主屏幕展示”(没有引号)。一定要包括你是如何制作它的描述，这样我们就可以把它作为下一个特色主屏幕。
