# 红色的大桌面

> 原文:[https://lifehacker.com/the-big-red-desktop-1609873040](https://lifehacker.com/the-big-red-desktop-1609873040)

有时候简单是最好的方式，虽然我们的很多桌面都有漂亮的壁纸和漂亮的外观，但这一款采用了简单的主题，并与它融为一体。它只显示最基本的信息，这就是 DeviantArt 用户 [sharmander3](http://sharmander3.deviantart.com/) 的全部需求。这是所有设置的方式。

Watch

如果你想在这里看到你的桌面精选，你应该提交到我们的 [**桌面展示区**](http://kinja.com/desktop-showcase) ！只要贴在你的 Kinja 博客上，标签为“桌面展示”，我们就能看到了。现在，这是你为自己重新创造这一个所需要的:

*   来自 DeviantArt 的 [白色条纹壁纸](http://cythean.deviantart.com/art/White-Stripe-128452312)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   sharmander3 为 Rainmeter 设计的 [大红色](http://sharmander3.deviantart.com/art/Big-Red-130781911) 皮肤，带有上图所示的 Launchy 皮肤、功率表、Gmail 监视器、时间、日期和温度显示。
*   屏幕右侧的 Stardock 的 [Objectdock 的](http://www.stardock.com/products/objectdock/)
*   Objectdock 中小图标的 [Amana 图标](http://imrik.deviantart.com/art/AMANA-ICONS-77919698) 包
*   屏幕左侧的
*   将 [倒影图标设置为](http://rocketdock.com/addon/icons/20322) 为火箭坞
*   记事本是 [Moxx Pack Notes](http://albinozz.deviantart.com/art/Moxx-Pack-126166622) 的修改版，但是 config 文件在上面的大红色下载中

它有许多可移动的部件，但最终产品看起来很棒，我们知道这需要很多工作才能实现。谢天谢地，sharmander3 把所有东西都放到了上面的雨量计配置 ZIP 中，这样我们其他人就可以抢先一步了。我们实际上是通过 [提交到我们的 Flickr 池](https://www.flickr.com/photos/47329900@N08/9448735202/in/pool-lifehacker-desktop-showandtell) 的另一个桌面偶然发现这个桌面的，但是更喜欢这个。
