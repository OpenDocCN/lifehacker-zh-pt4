# 启动管理器的四倍显示工作区

> 原文：<https://lifehacker.com/the-startup-managers-quadruple-display-workspace-1693371164>

当你在管理一家由十几个远程团队成员和三十多个实时产品组成的初创公司时，你所拥有的技术会让一切变得不同。该工作站配备了同时运行大量应用程序和任务的设备。



这个工作空间绝对是一项投资。有四个平铺的 34 英寸 LG 曲面超宽显示器，Mac Pro 和 MacBook Pro，两个 Wacom 图形平板电脑和 KAB 控制器椅。虽然这可能对我们许多人来说有些过头了，但对其他人来说，这可能是一个梦想，取决于你想做什么样的工作。该工作站属于 Teemu A .，他没有提到他/她管理的是哪家初创公司，也没有提到初创公司开发了哪些应用，但你可以在下面的《OS X 日报》采访中了解更多关于该工作站的信息。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间展示和讲述](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) Flickr 池中，与我们分享。确保您包含的任何照片至少为 640x360。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
