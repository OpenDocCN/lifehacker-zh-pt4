读者 [卡尔恩](http://caln.kinja.com/) 向我们的 [桌面展示区](http://kinja.com/tag/desktop-showcase) 提交了这个动画桌面，它是工作和娱乐的伟大结合——左边是工作空间和工具帮助，右边是一个有趣的动画频谱分析仪，与他的音乐相匹配。他是这样设置的。

Watch

如果你想在你的 Windows 机器上有同样的外观，这并不太难，而且与你所想的相反，Rainmeter 及其组件通常占用很少的系统资源，所以你可以在没有注意到它们在后台运行的情况下获得额外的好处。这是你需要的:

*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [桌面 VU 计](http://hitbit-pa.deviantart.com/art/Desktop-VU-Meter-2-0-515883194) 皮肤为雨量计为频谱分析仪
*   [雨量计](http://poiru.deviantart.com/art/Gnometer-1-3-0-3-182181512) 雨量计为系统监控小部件的皮肤
*   [YBoris 10 英尺 HUD](http://yboris.deviantart.com/art/10-Foot-HUD-YBoris-edit-105201352) 为雨量计创建时钟
*   [RocketDock](http://rocketdock.com/) 作为应用程序启动器出现在屏幕上方
*   [Stacks Docklet](http://rocketdock.com/addon/docklets/1791) 为图标“Stacks”，出现在上面视频中的一个图标上
*   [经典外壳](http://www.classicshell.net/) 自定义 Windows 8 开始菜单
*   [DropIt](http://www.dropitproject.com/) 在桌面上创建一个透明的回收站小部件来拖放文件

这里缺少了一些东西，比如特定的 dock 图标和壁纸，但考虑到你可以使用任何图标或任何你喜欢的壁纸来获得相同的效果，它们并不太重要。如果你有问题或者想知道更多关于 Caln 是如何设置的，请点击他的 Kinja 博客，链接如下！
