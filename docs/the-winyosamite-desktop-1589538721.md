# 温约塞米蒂桌面

> 原文：<https://lifehacker.com/the-winyosamite-desktop-1589538721>

读者 [赛法利](http://isaifalii.kinja.com/) 上周看到了苹果公司推出的 OS X“约塞米蒂”，他想把至少一些外观带到他的 Windows 机器上。合适的壁纸和一个小雨量计就足够了。他是这样设置的。



赛法利将他的桌面提交到了我们的 [**桌面展示区**](http://kinja.com/desktop-showcase) ，如果你想在这里看到你的桌面展示，这是你应该做的！你所需要做的就是把一个截图贴在你的 Kinja 博客上(你的评论账号附带的那个)，同时附上你用来定制它的壁纸、小工具和其他工具。

如果你想在你的 Windows 机器上复制这个，下面是你需要的:

*   [约塞米蒂壁纸](http://cdn.gottabemobile.com/wp-content/uploads/2014/06/Yosemite-Mac.jpg)T3】
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计的 [英格玛](http://kaelri.deviantart.com/art/Enigma-103823591) 皮肤创建右侧“通知中心”侧边栏，带有正在播放、系统状态、天气和日历小部件
*   [门户 2 桌面主题](http://wistfulwriter.deviantart.com/art/The-Portal-2-Desktop-Theme-for-Rainmeter-245722598) 为雨量计左上角为时钟和日期

这就是你所需要的。它简单明了，不需要太多的皮肤来定制。我们认为成品看起来很坚固，但当然，如果你对如何设置它或任何组件如何工作有疑问，请前往 Saifali 的 Kinja 博客(链接如下)提出你的问题或赞美你。

你有一台自己的好看又实用的台式机可以炫耀吗？分享给我们吧！**使用标签** [**将它发布到您的个人 Kinja 博客上**](http://kinja.com/tag/desktop-showcase) 或将其添加到我们的 [**Lifehacker 桌面展示并告诉 Flickr pool**](http://www.flickr.com/groups/lifehacker-desktop-showandtell/) 。截图必须至少为 640x360，并请包括您使用的信息，链接到您的壁纸，皮肤和主题，以及任何其他相关的细节。如果你的超棒的桌面吸引了我们的眼球，你可能会成为我们的特色！
