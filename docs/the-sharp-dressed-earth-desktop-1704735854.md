# 整洁的地球桌面

> 原文:[https://life hacker . com/the-sharp-dressed-earth-desktop-1704735854](https://lifehacker.com/the-sharp-dressed-earth-desktop-1704735854)

Flickr 的托马斯 将我最喜欢的一张壁纸与一些智能小工具和音乐播放控件结合起来，创建了一个工作空间，使用起来可能和看起来一样有趣。他是这样设计的。

Watch

这并不明显，但托马斯在这个桌面上运行 Windows，虽然你可能会从其他特色桌面上认出它(甚至是我的 GeekTool 指南)但他使用 Rainmeter 来完成所有事情。如果你也想这样做，你需要做的是:

*   [壁纸](http://www.icanbecreative.com/res/freeweb/2011/04/Earth.png)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计的 [Muziko 主题](http://nomnuggetnom.deviantart.com/art/Muziko-for-Rainmeter-314928622) 创建音乐播放控件和“正在播放”视图
*   [为雨量计编码主题](http://lilshizzy.deviantart.com/art/Rainmeter-Encoded-v1-2-3-215992941) 创建日期表头
*   [线 2.0 主题为雨量计](http://rasylver.deviantart.com/art/Lines-2-0-154719861) 为 RAM 和 CPU 监视器
*   雨量计的 [MemoDijay 皮肤](http://dijaysazon.deviantart.com/art/MemoDijay-s-Suite-213346436)Gmail 通知程序和时钟的

这就是这个的全部内容。最终产品看起来很棒，整个作品与壁纸的主题非常契合。他还指出，他定制了一些主题来改变字体和增加文本大小，并做了一些其他微妙的调整。要了解更多，请访问他的 Flickr 页面(链接如下)。)
