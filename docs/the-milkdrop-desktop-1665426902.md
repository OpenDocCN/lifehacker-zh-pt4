# drop 桌面

> 原文：<https://lifehacker.com/the-milkdrop-desktop-1665426902>

Flickr 用户 [LostWatson](https://www.flickr.com/photos/azeron1786/) 的桌面超现实而美丽，但也井井有条。边上的一个简单的启动器提供了对他的应用程序的访问，剩下的就是整洁和清晰了。这是他用来设置这一切的。

Watch

这个很简单，没有太多的组件。LostWatson 运行的是 Windows，所以下面是他使用的:

*   [壁纸](http://www.1wallpaperhd.com/2014/08/blue-color-background-wallpapers-18.html)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计的 [Omnimo UI](http://omnimo.info/)

任务栏被设置为自动隐藏，LostWatson 这样描述配置的其余部分:

> 我编辑了脚本，把所有东西都放在一个页面上，去掉了不必要的垃圾，并为类别添加了图片。RSS 提要和系统监视器会很好，但我还没有完全弄清楚作者是如何利用原始脚本中的隐藏功能的。

这就是全部了。雨量计和一个 UI 运行在它旁边。毫无疑问，这需要一些定制工作，如果你不熟悉它，可能会有点令人生畏。如果你有关于如何设置一切或使其工作的问题，点击下面的 Flickr 链接访问 LostWatson 的页面，并询问他是如何实现这一切的。你走的时候一定要让他知道是我们派你去的，我们喜欢他的作品！
