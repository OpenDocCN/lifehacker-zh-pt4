# 爆发主屏幕

> 原文：<https://lifehacker.com/the-outbreak-home-screen-1617906764>

在 Themer 发布后，我们最近在我的彩色屏幕上找到了一些乐趣。今天，我们带着离经叛道的艺术用户[revestianierange](http://revestianieorange.deviantart.com/)的爆发主屏幕回到我们的本源。



这个主屏幕减少了主屏幕的混乱，简化了界面的功能，而不是一堆乱七八糟的图标。一个页面是音乐播放器，另一个页面是统计数据，主页面显示电话、信息和应用程序等任务。这是他用来建造它的东西:

*   [新星发射器](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher)T3】
*   [Zooper Widget Pro](https://play.google.com/store/apps/details?id=org.zooper.zwpro)
*   [媒体实用程序](https://play.google.com/store/apps/details?id=com.batescorp.notificationmediacontrols.alpha)
*   [Zooper Widget 文件](https://www.dropbox.com/sh/w8rjdk55ji0dmka/o11Rph34VH)

你可以在下面的源代码链接中找到如何把它们放在一起的说明。

你有没有一个很棒的、被调整到遗忘的家或者你自己的锁屏，你想分享吗？把它贴在下面的评论里，或者贴在你自己的 Kinja 博客上，标签是“主屏幕展示”(没有引号)。一定要包括你是如何制作它的描述，这样我们就可以把它作为下一个特色主屏幕。
